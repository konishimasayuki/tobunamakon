# 東部生コン IME 自動切替 拡張機能

東部生コン業務システムの入力欄で、フィールドに応じて日本語IME(かな)/半角英数のヒントを送るブラウザ拡張機能です。

## インストール手順 (Chrome / Edge)

1. このフォルダ全体（`manifest.json` が入っているフォルダ）を**任意の場所に展開**しておきます
2. Chrome/Edge のアドレスバーに **`chrome://extensions`** と入力（Edge の場合は `edge://extensions`）
3. 右上の「**デベロッパーモード**」を **ON** にする
4. 「**パッケージ化されていない拡張機能を読み込む**」ボタンを押し、
   1. で展開したフォルダを選択する
5. 拡張機能一覧に「**東部生コン IME 自動切替**」が表示されれば完了

## 動作対象

- `https://tobunamakon.vercel.app/*`
- `https://*.tobunamakon.vercel.app/*` (プレビュー環境)
- `http://localhost:5173/*` (開発環境)

## モードの割当

- **全角かな(推奨)**: 業者名 / 商社名 / 現場名 / 現場住所 / 車種 / 打設箇所 / 荷下ろし / 特記 / 備考
- **半角英数(推奨)**: 受注日 / 日付 / 時間 / 配合 / 量 / 連絡先 / 現場連絡先

初回フォーカス時のみヒントを送ります。ユーザーが手動で IME 切替した後は上書きしません。

## 注意 / 制限事項

⚠️ **Windows の IME モードを Web ページ / 拡張機能から確実に切り替える標準的な方法は存在しません**。この拡張機能は以下のヒントを送るベストエフォート実装です:

- `lang` 属性の設定 (`ja` / `en`)
- `CompositionEvent` の dispatch
- `inputmode` 属性の設定

Windows + Chrome / Edge の Microsoft IME 環境で最大限効くよう作っていますが、環境や IME バージョンにより動作しない場合があります。

**完全な IME 制御には Native Messaging companion(Windows exe) の実装が必要**です。ご要望に応じて追加実装可能です。
